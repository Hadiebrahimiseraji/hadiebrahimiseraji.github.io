## نحوه آپلود به GitHub:

### روش 1: از طریق وب‌سایت GitHub
1. به https://github.com/Hadiebrahimiseraji/hadiebrahimiseraji.github.io بروید
2. به پوشه `assets/images/` بروید
3. روی دکمه "Add file" > "Upload files" کلیک کنید
4. فایل‌ها را بکشید و رها کنید
5. "Commit changes" را بزنید

### روش 2: از طریق Git (در ترمینال)
```bash
cd /workspaces/hadiebrahimiseraji.github.io
git add assets/images/
git commit -m "Add images"
git push
```
